Como utilizar o programa

Este programa permite que você registre dados de inspeção e gere relatórios em formato Excel.

Passos para usar:

  1. Preencha os campos do formulário com as informações necessárias.
  2. Os campos N da ordem, Especialidade, LT devem ser preenchidos só na primeira vez.
  3. Após enviar os dados, nos proximos prenchimentos preecher somente a partir do N da Torre.
  4. Clique no botão "Enviar Dados" para salvar os dados localmente.
  5. Para gerar a planilha, clique no botão "Gerar Planilha Inspeção".
  6. Se desejar finalizar a inspeção depois que gerou a planilha e limpar os dados, clique no botão "Finalizar Inspeção".

Se precisar de ajuda, entre em contato com o suporte técnico.
